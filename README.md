# Permission Protocol — Documentation

Technical specification, playbooks, and advanced examples for Permission Protocol.

> **No receipt = unauthorized autonomy.**

## Install the SDK

```bash
npm i @permission-protocol/sdk
```

https://www.npmjs.com/package/@permission-protocol/sdk

## Run the demo (10 minutes)

```bash
cd reference-demo
pnpm i
pnpm demo:run --all
```
